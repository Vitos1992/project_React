# Movie/Series Explorer

Легкий React-проєкт для перегляду популярних фільмів через TMDB API.

## Особливості

- Список популярних фільмів
- По 10 фільмів на сторінку
- Кожен фільм показує: назву, режисера та рік випуску
- Окрема сторінка фільму з детальною інформацією
- Реалізовано на React, хуки, компоненти та props
- Використовується React Router для маршрутизації

## Встановлення

1. Скопіюйте `.env.example` у `.env`
2. Додайте ключ TMDB API у `.env`:

```text
VITE_TMDB_API_KEY=YOUR_TMDB_API_KEY
```

3. Встановіть залежності:

```bash
npm install
```

4. Запустіть локально:

```bash
npm run dev
```

## Примітка

Проєкт підлаштований під умову без авторизації користувача. Ви потребуєте лише API-ключ TMDB для отримання даних через гостинний режим у додатку.
